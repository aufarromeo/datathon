# Panen.AI: Indonesia Agricultural Production Forecast
## 📌 Overview
Panen.AI is a data-driven web dashboard designed to enhance food security planning across Indonesia by providing highly accurate agricultural production forecasts[cite: 1]. It empowers policymakers, agricultural planners, and regional stakeholders to make informed decisions by combining advanced time-series forecasting with Explainable AI (XAI) and rigorous risk assessment models[cite: 1].

## 🚀 Key Features
$Real-Time Interactive Dashboard$: Monitor key metrics including Total Production, Forecast Growth, Model Accuracy, and Yield per Hectare across multiple commodities (Rice Paddy, Corn, Soybean, Cassava, Peanut, Sugarcane)[cite: 1].

$Multi-Model Forecasting$: Compare projections using a robust suite of statistical and machine learning models, including ARIMA, SARIMA, Prophet, Random Forest, LSTM Neural Networks, and Ensemble methods[cite: 1].

$Explainable AI (SHAP)$: Transparent predictions that break down the positive and negative drivers of crop yield (e.g., Rainfall, Temperature, Soil Moisture vs. Drought Index and Pest Pressure) using SHAP feature importance and waterfall charts[cite: 1].

$Extreme Value Risk Analysis$: Advanced probabilistic risk assessment using Generalized Extreme Value (GEV) distributions to calculate expected losses across various return periods (2-yr to 200-yr events)[cite: 1].

$Climate Scenario Mapping$: Evaluates the annual exceedance probability and yield loss impact of extreme events like Droughts, Floods, Pest Outbreaks, and Extreme Heat[cite: 1].

## 💻 Tech Stack
$Frontend$: HTML5, CSS3 (Custom Variables & Grid/Flexbox layouts), Vanilla JavaScript[cite: 1].

$Data Visualization$: Chart.js for responsive line, bar, and custom SHAP/Waterfall visual representations[cite: 1].

$Cloud Infrastructure & AI Backend$: Microsoft Azure (Hosting, Data Storage, and Model Deployment).

$Machine Learning Environment$: R and Python (Scikit-Learn, LightGBM, XGBoost, SHAP) utilized for model training and hyperparameter optimization before deployment.
